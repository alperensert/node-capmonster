# Introduction

[Capmonster.cloud](https://capmonster.cloud) is a cloud service for automatic recognition
of reCAPTCHA, hCaptcha and other types of captchas.
node-capmonster is a package for NodeJS.
Which allows you to basically use capmonster.cloud in your NodeJS projects.

## Available Captcha Types

-   Fun captcha
-   Geetest
-   HCaptcha
-   Image to text
-   ReCaptchaV2 Enterprise
-   ReCaptchaV2
-   ReCaptchaV3
-   Turnstile
-   FunCaptcha
