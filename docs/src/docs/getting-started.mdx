## Installation

#### Using yarn:

```bash
yarn add node-capmonster
```

#### Using npm:

```bash
npm i node-capmonster
```

#### Using pnpm:

```bash
pnpm add node-capmonster
```

## Dive into code

Select a captcha type from the menu and see usage examples.
